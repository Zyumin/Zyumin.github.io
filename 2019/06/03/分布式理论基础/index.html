从ACID 到CAP/ BASE

传统DBMS 中的ACID

  • Atomicity 原子性
  • Isolation 隔离线
    • 未授权读取(RU);
    • 授权读取(RC);
    • 可重复读取(RR);
    • 串行化(S)
  • Consistency 一致性
  • Durability 持久性

​

CAP

  • A 高可用
  • C 一致性
  • P 分区 副本

在C和A之中寻找平衡

BASE

  • Basically Acaliable
    • 响应时间损失
    • 功能上降级
  • Soft state
  • Eventually consistent

不止分布式环境,现代数据库主从复制
